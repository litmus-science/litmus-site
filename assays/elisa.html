AI-Readiness

Data your models can actually use

ELISA results from Litmus arrive pre-processed and structured — not as a PDF or raw plate reader export. Every result includes the fitted curve, calculated concentrations, and full experimental metadata in a single JSON payload.

  • 4-parameter logistic (4PL) curve fitting applied automatically to every standard curve
  • LOD and LOQ calculated and flagged per assay run
  • Interplate normalization when running multi-plate experiments
  • Raw OD values included alongside processed concentrations
  • Full metadata: antibody lot, plate map, operator ID, timestamp
