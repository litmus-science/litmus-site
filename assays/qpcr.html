AI-Readiness

Gene expression data your models can ingest directly

qPCR data from Litmus is delivered pre-normalized and structured. Multiple reference genes, full plate layout metadata, and normalized fold-change values arrive in a single JSON payload — no manual Cq export or Excel formula work required.

  • Raw Cq values for every well in the plate layout
  • ΔΔCt and fold-change calculated with multiple reference genes
  • Reference gene stability scoring (geNorm/NormFinder)
  • Amplification efficiency per primer pair
  • 384-well plate map included for spatial normalization
